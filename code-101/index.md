# Code 101 - Elementals

**Duración**: 6 semanas (54 horas)  
**Modalidad**: Clases en vivo online (2 veces por semana, 8:00 PM - 10:30 PM)  
**Dirigido a**: Personas adultas sin experiencia previa en programación, con habilidades digitales básicas.

Este curso es tu primer paso hacia convertirte en desarrollador(a). Aprenderás cómo funciona la web moderna, cómo construir páginas accesibles con HTML y CSS, y cómo resolver problemas lógicos con JavaScript. Además, dominarás herramientas esenciales como VS Code, la terminal, GitHub y asistentes con IA como ChatGPT y Copilot.

> 🟢 No se necesitan conocimientos previos.

## 🎯 Objetivos de Aprendizaje

Al finalizar este curso, podrás:

- Comprender cómo funciona una aplicación web moderna.
- Crear estructuras accesibles con HTML semántico.
- Aplicar estilos y layouts básicos con CSS.
- Usar herramientas de desarrollo (VS Code, Terminal, Git/GitHub).
- Implementar lógica de programación con JavaScript.
- Trabajar de forma colaborativa con herramientas de Inteligencia Artificial.
- Presentar un proyecto técnico básico con solución implementada.

## Sílabo del Curso

### ▶︎ Módulo 1: Introducción al Desarrollo Web
En este módulo inicial, nos sumergiremos en los fundamentos del desarrollo web moderno. Aprenderás a crear páginas web accesibles y bien estructuradas utilizando HTML semántico, mientras dominas los conceptos básicos de estilizado con CSS. Comprenderás el flujo de trabajo de una aplicación web desde su concepción hasta su despliegue, y aprenderás a documentar y compartir tu trabajo usando Markdown y GitHub Pages.
- _Mini Proyecto_: **Sitio web estilizado en la nube**.

### ▶︎ Módulo 2: Herramientas del Desarrollador
Este módulo te equipará con las herramientas esenciales que todo desarrollador moderno necesita dominar. Desde la configuración de tu entorno de desarrollo hasta el uso efectivo de asistentes de IA, aprenderás a trabajar de manera más eficiente. Explorarás el diseño web responsive y las herramientas de desarrollo del navegador, mientras desarrollas habilidades de wireframing para crear interfaces intuitivas y atractivas.
- _Mini Proyecto_: **Hub Personal de Redes Sociales**.

### ▶︎ Módulo 3: Introducción a la Programación con JavaScript
En este módulo final, te introducirás al mundo de la programación a través de JavaScript. Aprenderás los conceptos fundamentales de la programación, desde variables y tipos de datos hasta funciones y lógica condicional. Desarrollarás tu pensamiento algorítmico y aprenderás a resolver problemas de manera estructurada, culminando en la creación de un proyecto interactivo que demostrará todas tus habilidades adquiridas.
- _Proyecto final_: **Juego Interactivo con Javascript**.
